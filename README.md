# [Onironautes.fr](https://onironautes.fr)

## Local development

Requires Ruby >= 3.

Launch a server on http://localhost:8080 with:

```sh
bin/dev
```

Call build every time you change the source files:

```sh
bin/build
```

## Watch and rebuild

Requires `overmind` and `npm install -g nodemon`:

```sh
bin/watch
```

## Deploy

Commit using git, then:

```sh
bin/deploy
```
