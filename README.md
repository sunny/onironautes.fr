# [Onironautes.fr](https://onironautes.fr)

## Local development

Requires Ruby >= 3, `overmind` and `npm install -g nodemon`.

Launch a server on http://localhost:8080 and rebuild with:

```sh
bin/dev
```

## Deploy

Make sure you have the latest build:

```sh
bin/build
```

Commit using git, then:

```sh
bin/deploy
```
